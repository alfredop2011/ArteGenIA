5. Settings → Channel → activar "Made for kids: No"

### Profile pic

- **Tamaño**: 800×800px (YouTube lo redimensiona a círculo)
- **Contenido**: logo ArteGenIA

### Channel banner (CRÍTICO — el que más se ve)

- **Tamaño exacto**: 2560×1440px
- **Safe area** (visible en mobile): centrado 1546×423px
- **Contenido sugerido**:
  - Lado izquierdo: gradiente púrpura-fucsia con texto "EDITOR FLYERS DJ CON IA"
  - Lado derecho: collage de 3-4 plantillas DJ tuyas (Techno Dark, Reggaeton, Vinyl Vintage)

### Cómo crear el banner (DOGFOODING — usa tu propio editor)

1. Entra a [artegenia.com/create](https://artegenia.com/create) (cuando esté listo)
2. O usa una plantilla con formato "fb-cover" (1920×1005, cerca del ratio YT)
3. Adapta texto + colores
4. Exporta JPG y reescala a 2560×1440 en Photoshop / Photopea / Canva

Alternativa rápida: usa Canva → template "YouTube Banner" → reescribe texto.

Pinterest es **buscador visual**, no red social. Funciona así:
1. Crear "boards" temáticos: "Flyers DJ Techno", "Carteles Bachata", "Posters Festival"
2. Pinear thumbnails de tus plantillas con descripciones SEO ("Plantilla flyer DJ techno gratis")

4. Los pins ranquean en Google Images automáticamente

**ROI bajo a corto plazo, alto a 6+ meses** (SEO toma tiempo). Solo dedica 1 hora/mes a subir pins.

## Plan de creación de assets visuales

Necesitas estos 7 assets:

| Asset | Tamaño | Para qué |
|---|---|---|
| Logo cuadrado 200×200 | 200×200 | TikTok profile |
| Logo cuadrado 400×400 | 400×400 | Twitter, Instagram |
| Logo cuadrado 800×800 | 800×800 | YouTube profile |
| Banner YouTube | 2560×1440 | YouTube canal |
| Banner Twitter | 1500×500 | Twitter header |
| Cover LinkedIn | 1128×191 | LinkedIn page header |
| Pinterest pin template | 1000×1500 | Pins individuales |

### Cómo crearlos (DOGFOODING + Canva)

**Approach recomendado**: usa tu propio ArteGenIA para los banners (caso real para vídeo TikTok #1 "uso mi propio producto") + Canva para reescalar a tamaños exactos:

1. **Logo**: usa el que ya tienes en el header de artegenia.com (descárgalo PNG transparente desde DevTools)
2. **Banners** (YouTube, Twitter, LinkedIn):
   - Crea en ArteGenIA con formato "fb-cover" (cerca del ratio)
   - Exporta JPG
   - Reescala en Canva o Photopea al tamaño exacto de cada plataforma
3. **Pinterest pins**: usa directamente las plantillas DJ (format portrait 1080×1350) que ya tienes

### Alternativa más rápida

Si no tienes tiempo para dogfooding ahora:
- Canva → templates "YouTube Banner", "Twitter Header", "LinkedIn Cover"
- Reescribe texto y colores a tu paleta
- 20 min total
