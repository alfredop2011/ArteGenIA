El editor se adapta a TODA tu industria de eventos

El mismo editor mobile sirve para conciertos, fiestas, festivales, galas, clubs, eventos corporativos y clases. Lo que cambia son los bloques editables — cada plantilla del catálogo define los suyos.

CLAVE

El editor es genérico, las plantillas son específicas. Cuando el usuario abre una plantilla de "Concierto", verá bloques como Artista, Fecha, Precio anticipada/puerta. Cuando abre una de "Gala", verá Dress code, RSVP, Confirmación. La UI nunca cambia, solo el contenido del form.
